## To run this project locally you need to clone the repo

### Clone project
```
git clone git@github.com:TangoPJ/vue-table.git
```

### Go to repo directory
```
cd vue-table
```

### Then install dependencies:
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```
